## Overview
Each feature in the project modals can be clicked to display a full-screen screenshot. This guide explains how to add your own feature screenshots.

## Image Location
All feature screenshots should be placed in: `images/features/`

## Image Specifications
- **Format**: PNG (recommended) or JPG
- **Dimensions**: 1920x1080px or higher (16:9 aspect ratio recommended)
- **File Size**: Keep under 500KB for optimal loading
- **Theme**: Dark theme images work best with the site design

## How to Add Images

1. Take or create screenshots of each feature
2. Save them with the exact filenames listed above
3. Place them in the `images/features/` directory
4. The website will automatically use them when features are clicked

## Notes
- If an image is missing, the feature will still show but won't open the lightbox
- You currently have 1/36 images (reservation_system.png)
- Image generation quota resets in ~4 hours if you want AI-generated images
